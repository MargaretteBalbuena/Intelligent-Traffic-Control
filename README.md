# Intelligent-Traffic-Control

Vanilla HTML, CSS, and JavaScript dashboard that simulates an AI-assisted traffic control system across two intersections.

## Features

- Real-time adaptive signal simulation for North Gate and Central Square
- AI-assisted timing mode with fixed-time fallback for comparison
- Live network KPIs, event trace, and queue pressure indicators
- Fully responsive dashboard UI with no frameworks or build tooling

## Run

Open `index.html` in a browser.
